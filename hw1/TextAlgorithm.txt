1. Go to booking.com
2. Check if user is already logged in
    2.1. If yes, go to 'enter search criteria field'
    2.2. If no, check if user has an account
        2.2.1. If user has an account: sign in
            2.2.1.1. Go to 'enter search criteria field'
        2.2.2. Else sign up
            2.2.2.1. Go to 'enter search criteria field'
3. Choose appropriate variant
4. Ask user if he wants to book it
    4.1. If yes:
        4.1.1. Book variant
        4.1.2. Save string 'Booking succeed' to variable 'message'
        4.1.3. Message output
        4.1.4. End of program
    4.2. If no: 
        4.2.1. Check if there are any other variants
            4.2.1.1. If yes go back to 'enter search criteria field'
            4.2.1.2. If no save string 'There are no appropriate variants for you' to variable message
            4.2.1.3. Message output
            4.2.1.4. Go back to 'choose appropriate variant' field
